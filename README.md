# NestJS TypeScript TypeORM PostgreSQL Jest

## Project setup

```bash
$ npm install
```

```bash
# TO RUN LOCALLY
$ docker-compose up postgres
$ npm start

# OR

# TO RUN ON DOCKER
$ docker-compose up --build
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
